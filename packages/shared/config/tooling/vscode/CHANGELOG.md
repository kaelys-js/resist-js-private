# Changelog

All notable changes to the Resist Tooling extension will be documented in this file.

## [0.0.1] - 2026-03-31

### Added

- Real-time linting with configurable triggers (on type, on save, on open)
- Quick fix code actions with auto-fix suggestions
- "Fix All" action for batch auto-fixing
- Diff preview for reviewing fixes before applying
- Code lens showing rule IDs and issue counts inline
- Per-rule disable comments (line-level and file-level)
- Diagnostic filtering by category
- Performance timing breakdown via output channel
- Unused import removal command
- Build stage indicator with quick-pick switching
- Format-on-save integration via formatting provider
- Stale diagnostic cleanup for idle files
- Multi-root workspace support with per-folder configuration
- Config file watcher for automatic re-lint on config changes
- Notification deduplication for missing binary warnings
- Full locale system with parameterized string templates
- Brand constants for white-label support
- 306 unit tests with coverage thresholds
