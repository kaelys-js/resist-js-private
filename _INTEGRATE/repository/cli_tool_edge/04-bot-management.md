# 04 — Bot Management: Bot Fight Mode, Super Bot Fight Mode, Bot Scores, AI Bot Blocking

## Context

Bot Management is a critical Cloudflare security feature that classifies incoming requests as human, automated, or verified bot traffic. It assigns a **bot score** (1-99, where 1 = definitely bot, 99 = definitely human) and provides configurable actions based on classification tiers.

Locally, we simulate bot management using Caddy middleware that:
- Matches User-Agent strings against known bot pattern databases
- Assigns simulated bot scores based on UA + request heuristics
- Injects `CF-Bot-Score`, `CF-Is-Bot`, `CF-Verified-Bot` headers for Worker access via `request.cf.botManagement`
- Serves challenge pages or blocks based on configurable thresholds

The same Valibot schema drives both:
- **Local simulation** via Caddy header matching and injection (generated by `bot.ts`)
- **Production deployment** via Pulumi Cloudflare Bot Management API resources

### How Cloudflare Bot Management Works

1. Every request is analyzed using multiple signals: User-Agent, TLS fingerprint (JA3/JA4), behavioral patterns, IP reputation, and JavaScript detection results.
2. A **bot score** (1-99) is assigned: 1 = definitely automated, 99 = definitely human.
3. Traffic is classified into tiers:
   - **Definitely Automated** — Bot score 1-29
   - **Likely Automated** — Bot score 30-49
   - **Likely Human** — Bot score 50-99
4. **Verified bots** (Googlebot, Bingbot, etc.) are identified separately regardless of score.
5. Configurable actions per tier: Allow, Block, Challenge, Managed Challenge.
6. `cf.bot_management.*` fields are available in Workers and Rules Engine.

### Local Simulation Strategy

- **User-Agent matching** against known bot signature lists (crawlers, scrapers, automation tools).
- **Request heuristic analysis**: missing Accept-Language, missing Cookie on non-first request, uncommon Accept headers.
- **Verified bot identification**: Known search engine crawler UA patterns + optional DNS verification.
- **Score injection**: Computed score injected as `CF-Bot-Score` header → translated to `request.cf.botManagement.score` by wrangler middleware.

```typescript
/**
 * Bot Management → Caddy Directives Generator
 *
 * Generates Caddy configuration for bot detection, scoring,
 * header injection, and action enforcement.
 *
 * @module
 */

import * as v from 'valibot';
import { safeParse } from '@/utils/result/safe';
import type { Result } from '@/utils/result/types';
import { ok, err } from '@/utils/result/helpers';
import { BotManagementConfigSchema } from '@/schemas/core-config/src/edge-security';
import type { BotManagementConfig } from '@/schemas/core-config/src/edge-security';
import {
	VERIFIED_BOT_PATTERNS,
	AUTOMATED_BOT_PATTERNS,
	AI_CRAWLER_PATTERNS,
	HUMAN_SIGNAL_HEADERS,
} from './bot-patterns';

// =============================================================================
// Bot Score Calculation Logic (Caddy CEL/Matcher Expressions)
// =============================================================================

/**
 * Generate Caddy matchers for verified bot detection.
 *
 * Creates named matchers that identify known verified bots
 * by User-Agent pattern.
 *
 * @param config - Bot management configuration.
 * @returns Result containing Caddy matcher directive strings.
 */
export function generateVerifiedBotMatchers(
	config: BotManagementConfig,
): Result<ReadonlyArray<v.InferOutput<typeof v.string>>> {
	const directives: Array<string> = [
		'# Verified Bot Detection Matchers',
	];

	// Combine built-in and custom verified bots
	const allPatterns = [...VERIFIED_BOT_PATTERNS];
	if (config.customVerifiedBots && config.customVerifiedBots.length > 0) {
		for (const custom of config.customVerifiedBots) {
			allPatterns.push({
				name: custom.name,
				pattern: custom.pattern,
				category: custom.category,
			});
		}
	}

	const verifiedPattern = allPatterns.map((b) => b.pattern).join('|');
	directives.push(`@verified_bot header_regexp User-Agent "(?i)(${verifiedPattern})"`);
	directives.push('');

	return ok(directives);
}

/**
 * Generate Caddy matchers for automated bot detection.
 *
 * Creates named matchers that identify known automated bots
 * and suspicious request patterns.
 *
 * @returns Result containing Caddy matcher directive strings.
 */
export function generateAutomatedBotMatchers(): Result<ReadonlyArray<v.InferOutput<typeof v.string>>> {
	const directives: Array<string> = [
		'# Automated Bot Detection Matchers',
	];

	const automatedPattern = AUTOMATED_BOT_PATTERNS.join('|');
	directives.push(`@definitely_automated header_regexp User-Agent "(?i)(${automatedPattern})"`);
	directives.push('');

	// Heuristic: missing Accept-Language suggests non-browser
	directives.push('# Heuristic: missing typical browser headers');
	directives.push('@likely_automated {');
	directives.push('	not header Accept-Language *');
	directives.push('	not @verified_bot');
	directives.push('	not @definitely_automated');
	directives.push('}');
	directives.push('');

	return ok(directives);
}

/**
 * Generate Caddy matchers for AI crawler bot detection.
 *
 * @param config - Bot management configuration.
 * @returns Result containing Caddy matcher directive strings.
 */
export function generateAiCrawlerMatchers(
	config: BotManagementConfig,
): Result<ReadonlyArray<v.InferOutput<typeof v.string>>> {
	if (!config.aiBlocking?.enabled) {
		return ok([]);
	}

	const allPatterns = [...AI_CRAWLER_PATTERNS];
	if (config.aiBlocking.additionalPatterns && config.aiBlocking.additionalPatterns.length > 0) {
		allPatterns.push(...config.aiBlocking.additionalPatterns);
	}

	const aiPattern = allPatterns.join('|');
	const directives: Array<string> = [
		'# AI Crawler Bot Matchers',
		`@ai_crawler header_regexp User-Agent "(?i)(${aiPattern})"`,
		'',
	];

	return ok(directives);
}

/**
 * Generate Caddy header directives for bot score injection.
 *
 * Injects bot management headers based on request classification:
 * - `CF-Bot-Score` — Simulated bot score (1-99)
 * - `CF-Is-Bot` — Boolean bot detection
 * - `CF-Verified-Bot` — Boolean verified bot status
 * - `CF-Verified-Bot-Category` — Verified bot category string
 * - `CF-Threat-Score` — Threat score (0-100)
 *
 * @param config - Bot management configuration.
 * @returns Result containing Caddy directive strings for header injection.
 */
export function generateBotHeaderInjection(
	config: BotManagementConfig,
): Result<ReadonlyArray<v.InferOutput<typeof v.string>>> {
	const scores = config.scoreThresholds ?? {};
	const knownBotScore = scores.knownBot ?? 10;
	const suspiciousScore = scores.suspiciousHeuristic ?? 35;
	const normalScore = scores.normalHuman ?? 85;
	const verifiedBotScore = scores.verifiedBot ?? 1;

	const directives: Array<string> = [
		'# Bot Score Header Injection',
		'# Default: likely human',
		`header_up CF-Bot-Score "${String(normalScore)}"`,
		'header_up CF-Is-Bot "false"',
		'header_up CF-Verified-Bot "false"',
		'header_up CF-Verified-Bot-Category ""',
		`header_up CF-Threat-Score "0"`,
		'',
		'# Override: verified bot',
		'handle @verified_bot {',
		`	header_up CF-Bot-Score "${String(verifiedBotScore)}"`,
		'	header_up CF-Is-Bot "false"',
		'	header_up CF-Verified-Bot "true"',
		'	# Category is set per-bot pattern in extended matcher',
		'}',
		'',
		'# Override: definitely automated',
		'handle @definitely_automated {',
		`	header_up CF-Bot-Score "${String(knownBotScore)}"`,
		'	header_up CF-Is-Bot "true"',
		'	header_up CF-Verified-Bot "false"',
		`	header_up CF-Threat-Score "75"`,
		'}',
		'',
		'# Override: likely automated (heuristic)',
		'handle @likely_automated {',
		`	header_up CF-Bot-Score "${String(suspiciousScore)}"`,
		'	header_up CF-Is-Bot "true"',
		'	header_up CF-Verified-Bot "false"',
		`	header_up CF-Threat-Score "40"`,
		'}',
		'',
	];

	return ok(directives);
}

/**
 * Maps bot action to Caddy response directive.
 *
 * @param action - Bot management action from config.
 * @returns Caddy directive string for the action.
 */
function botActionToDirective(action: v.InferOutput<typeof v.string>): v.InferOutput<typeof v.string> {
	switch (action) {
		case 'block': return 'respond "Access Denied — Bot Detected" 403';
		case 'challenge': return 'respond "Challenge Required" 403';
		case 'managed_challenge': return 'respond "Challenge Required" 403';
		case 'allow':
		default: return '# allowed';
	}
}

/**
 * Generate Caddy action enforcement directives.
 *
 * Applies configured actions (block/challenge/allow) for each
 * bot traffic tier.
 *
 * @param config - Bot management configuration.
 * @returns Result containing Caddy directive strings for action enforcement.
 */
export function generateBotActionDirectives(
	config: BotManagementConfig,
): Result<ReadonlyArray<v.InferOutput<typeof v.string>>> {
	const sbfm = config.superBotFightMode ?? {};
	const directives: Array<string> = [];

	// AI crawler blocking (highest priority)
	if (config.aiBlocking?.enabled) {
		directives.push('# AI Crawler Blocking');
		directives.push('handle @ai_crawler {');
		directives.push('	respond "AI Crawlers Blocked" 403');
		directives.push('}');
		directives.push('');
	}

	// Definitely automated
	if (sbfm.definitelyAutomated && sbfm.definitelyAutomated !== 'allow') {
		directives.push('# Definitely Automated — action: ' + sbfm.definitelyAutomated);
		directives.push('handle @definitely_automated {');

		// Skip static resources if protection is off
		if (!sbfm.staticResourceProtection) {
			directives.push('	@not_static not @static_resource');
			directives.push('	handle @not_static {');
			directives.push('		' + botActionToDirective(sbfm.definitelyAutomated));
			directives.push('	}');
		} else {
			directives.push('	' + botActionToDirective(sbfm.definitelyAutomated));
		}

		directives.push('}');
		directives.push('');
	}

	// Likely automated
	if (sbfm.likelyAutomated && sbfm.likelyAutomated !== 'allow') {
		directives.push('# Likely Automated — action: ' + sbfm.likelyAutomated);
		directives.push('handle @likely_automated {');

		if (!sbfm.staticResourceProtection) {
			directives.push('	@not_static not @static_resource');
			directives.push('	handle @not_static {');
			directives.push('		' + botActionToDirective(sbfm.likelyAutomated));
			directives.push('	}');
		} else {
			directives.push('	' + botActionToDirective(sbfm.likelyAutomated));
		}

		directives.push('}');
		directives.push('');
	}

	// Verified bots
	if (sbfm.verifiedBots === 'block') {
		directives.push('# Verified Bots — blocked');
		directives.push('handle @verified_bot {');
		directives.push('	respond "Verified Bots Blocked" 403');
		directives.push('}');
		directives.push('');
	}

	return ok(directives);
}

/**
 * Generate JavaScript detection snippet injection.
 *
 * Injects a small JS snippet into HTML responses that sets a detection
 * cookie. Requests without the cookie on subsequent visits are flagged
 * as potentially headless/automated.
 *
 * @returns Caddy response filter directive strings.
 */
export function generateJsDetectionInjection(): ReadonlyArray<v.InferOutput<typeof v.string>> {
	const jsSnippet = `<script>document.cookie="cf_js_detect=1;path=/;max-age=1800;SameSite=Strict";</script>`;

	return [
		'# JavaScript Detection Injection',
		'# Inject JS snippet before </head> in HTML responses',
		'templates',
		`respond /cdn-cgi/challenge-platform/scripts/jsd/main.js 200 {`,
		`	body "${jsSnippet}"`,
		`	header Content-Type "application/javascript"`,
		'}',
		'',
	];
}

// =============================================================================
// Managed robots.txt Generation
// =============================================================================

/**
 * Generate robots.txt content from bot management config.
 *
 * @param config - Bot management configuration.
 * @returns Result containing the robots.txt content string.
 */
export function generateRobotsTxt(
	config: BotManagementConfig,
): Result<v.InferOutput<typeof v.string>> {
	const lines: Array<string> = [];

	// Block AI crawlers if AI blocking is enabled
	if (config.aiBlocking?.enabled) {
		for (const pattern of AI_CRAWLER_PATTERNS) {
			lines.push(`User-agent: ${pattern}`);
			lines.push('Disallow: /');
			lines.push('');
		}
	}

	// Add custom rules
	if (config.managedRobotsTxt?.additionalRules) {
		for (const rule of config.managedRobotsTxt.additionalRules) {
			lines.push(`User-agent: ${rule.userAgent}`);
			for (const path of rule.disallow) {
				lines.push(`Disallow: ${path}`);
			}
			lines.push('');
		}
	}

	// Default: allow all
	lines.push('User-agent: *');
	lines.push('Allow: /');
	lines.push('');

	return ok(lines.join('\n'));
}

// =============================================================================

/**
 * Generate complete bot management Caddyfile directive block.
 *
 * Composes all bot management sub-features into Caddy configuration
 * that integrates into the Caddyfile.
 *
 * @param config - Complete bot management configuration from edge config.
 * @returns Result containing the complete bot management directive block.
 */
export function generateBotManagementDirectives(
	config: BotManagementConfig,
): Result<v.InferOutput<typeof v.string>> {
	const configResult = safeParse(BotManagementConfigSchema, config);
	if (!configResult.ok) return configResult;
	const validConfig = configResult.data;

	if (!validConfig.enabled) {
		return ok('');
	}

	const sections: Array<string> = [];

	sections.push('# ==========================================================');
	sections.push('# Bot Management');
	sections.push('# ==========================================================');
	sections.push('');

	// Static resource matcher (needed for protection toggle)
	sections.push(...generateStaticResourceMatcher());

	// Verified bot matchers
	const verifiedResult = generateVerifiedBotMatchers(validConfig);
	if (!verifiedResult.ok) return verifiedResult;
	sections.push(...verifiedResult.data);

	// Automated bot matchers
	const automatedResult = generateAutomatedBotMatchers();
	if (!automatedResult.ok) return automatedResult;
	sections.push(...automatedResult.data);

	// AI crawler matchers
	const aiResult = generateAiCrawlerMatchers(validConfig);
	if (!aiResult.ok) return aiResult;
	if (aiResult.data.length > 0) {
		sections.push(...aiResult.data);
	}

	// Header injection (bot scores)
	const headerResult = generateBotHeaderInjection(validConfig);
	if (!headerResult.ok) return headerResult;
	sections.push(...headerResult.data);

	// Action enforcement (block/challenge)
	const actionResult = generateBotActionDirectives(validConfig);
	if (!actionResult.ok) return actionResult;
	if (actionResult.data.length > 0) {
		sections.push(...actionResult.data);
	}

	// JS detection injection
	if (validConfig.superBotFightMode?.jsDetection) {
		sections.push(...generateJsDetectionInjection());
	}

	// Managed robots.txt
	if (validConfig.managedRobotsTxt?.enabled) {
		const robotsResult = generateRobotsTxt(validConfig);
		if (!robotsResult.ok) return robotsResult;

		sections.push('# Managed robots.txt');
		sections.push('handle /robots.txt {');
		sections.push(`	respond "${robotsResult.data.replace(/"/g, '\\"')}" 200 {`);
		sections.push('		header Content-Type "text/plain"');
		sections.push('	}');
		sections.push('}');
		sections.push('');
	}

	// Bot analytics logging
	if (validConfig.enableAnalytics) {
		sections.push(...generateBotAnalyticsLog());
	}

	return ok(sections.join('\n'));
}
```

```typescript
/**
 * Pulumi Bot Management Resource Generator
 *
 * Maps BotManagementConfig → Cloudflare Bot Management API resources.
 *
 * @module
 */

import * as pulumi from '@pulumi/pulumi';
import * as cloudflare from '@pulumi/cloudflare';
import type { BotManagementConfig } from '@resist/schemas/core-config/src/edge-security';
import type { Result } from '@/utils/result/types';
import { ok } from '@/utils/result/helpers';

/**
 * Create Cloudflare Bot Management resources from config.
 *
 * @param zoneId - Cloudflare zone ID.
 * @param config - Bot management configuration.
 * @param namePrefix - Resource name prefix.
 * @returns Result containing the bot management resources.
 */
export function createBotManagementResources(
	zoneId: pulumi.Input<v.InferOutput<typeof v.string>>,
	config: BotManagementConfig,
	namePrefix: v.InferOutput<typeof v.string>,
): Result<{
	botManagement: cloudflare.BotManagement;
	customRuleset?: cloudflare.Ruleset;
}> {
	// Bot Management zone setting
	const botManagement = new cloudflare.BotManagement(`${namePrefix}-bot-mgmt`, {
		zoneId,
		enableJs: config.superBotFightMode?.jsDetection ?? false,
		fightMode: config.botFightMode?.enabled ?? false,
		sbfmDefinitelyAutomated: config.superBotFightMode?.definitelyAutomated ?? 'block',
		sbfmLikelyAutomated: config.superBotFightMode?.likelyAutomated ?? 'managed_challenge',
		sbfmVerifiedBots: config.superBotFightMode?.verifiedBots ?? 'allow',
		sbfmStaticResourceProtection: config.superBotFightMode?.staticResourceProtection ?? false,
		optimizeWordpress: config.superBotFightMode?.optimizeForWordpress ?? false,
	});

	const resources: {
		botManagement: cloudflare.BotManagement;
		customRuleset?: cloudflare.Ruleset;
	} = { botManagement };

	// AI bot blocking via custom WAF rule (uses `cf.bot_management.verified_bot` field)
	if (config.aiBlocking?.enabled) {
		resources.customRuleset = new cloudflare.Ruleset(`${namePrefix}-ai-bot-block`, {
			zoneId,
			name: `${namePrefix} AI Bot Blocking`,
			description: 'Block known AI crawler bots',
			kind: 'zone',
			phase: 'http_request_firewall_custom',
			rules: [{
				action: 'block',
				expression: '(cf.bot_management.verified_bot) and (cf.verified_bot_category eq "AI Crawler")',
				description: 'Block AI crawler bots',
				enabled: true,
			}],
		});
	}

	return ok(resources);
}
```

## 8. Known Limitations & Simulation Gaps

| Feature | Cloudflare | Local Simulation | Gap |
|---------|-----------|-----------------|-----|
| Bot score ML model | ML-based on 40+ signals | UA regex + heuristics | Much less accurate |
| TLS fingerprinting (JA3/JA4) | Real TLS handshake analysis | Not available locally | No fingerprint data |
| IP reputation | Global IP threat database | Not available locally | No IP reputation |
| Behavioral analysis | Cross-request pattern analysis | Single-request heuristics only | No behavioral patterns |
| JavaScript detection | Real browser environment testing | Cookie-based detection | Easily spoofable |
| Turnstile integration | Full CAPTCHA challenge flow | Static 403 response | No interactive challenge |
| Verified bot DNS verification | Reverse DNS lookup validation | UA pattern matching only | Can be spoofed |
| Bot score precision | 1-99 with ML precision | Fixed scores per category | Coarse scoring |
